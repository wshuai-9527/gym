Gym Vault V29 Cloud 上传说明

1. 解压本压缩包。
2. GitHub 仓库 gym 根目录中，删除旧的 index.html / sw.js / manifest.webmanifest / icon.svg / images 文件夹。
3. 上传本包内所有文件到仓库根目录。
4. GitHub Pages 打开：
   https://wshuai-9527.github.io/gym/?v=29&clean=1
5. 进入“档案”页，输入邮箱，点“发送登录链接”。
6. 邮箱登录成功后，点“上传本机档案”，会把当前历史写入 Supabase。
7. 以后保存每日训练，会自动同步云端。

注意：Supabase 只使用 publishable key，没有写入 secret/service_role key。
